#### 👤 [Facial Books](https://github.com/Thepenguin07/facial-books)
* **Core Focus:** Computer Vision, One-Shot Metric Learning, & Real-Time Payroll Automation
* **Core Algorithmic Architecture:** Replaces manual check-in systems using a custom Siamese Neural Network trained from scratch for real-time facial verification via webcam feed. By using a One-Shot Metric Learning approach, the network scales cleanly to new faces without requiring constant retraining or risk of overfitting on small sample sizes.
* **Optimization & Loss Formulations:** Transforms raw spatial face configurations into an L2-normalized 128-dimensional embedding space. Optimization is achieved via a custom implementation of **Contrastive Loss**, passing image pairs across a margin of $m = 1.0$:
  $$L_{\text{Contrastive}} = \frac{1}{2}(Y)(D_W)^2 + \frac{1}{2}(1-Y)\max(0, m - D_W)^2$$
  Where the Euclidean distance $D_W$ is computed as:
  $$D_W(X_1, X_2) = \|G_W(X_1) - G_W(X_2)\|_2$$
  A robust runtime decision boundary threshold of $\tau = 0.12$ regulates employee authorization, entry/exit timestamping, and automated payroll allocations.
* **Neural Network Topology:** Processes $96 \times 96 \times 3$ normalized RGB tensor blocks through 4 structural convolutional blocks scaling through 32, 64, 128, and 256 filters (`Conv2D` $\rightarrow$ `BatchNormalization` $\rightarrow$ `ReLU` $\rightarrow$ `MaxPooling` $\rightarrow$ `Dropout`). Features collapse into a dense 128-dimensional vector layer passed through custom-subclassed `L2Normalize` layers for native serialization in Keras.

#### 🕸️ [Social Graph](https://github.com/Thepenguin07/socialgraph)
* **Core Focus:** Discrete Mathematics, Graph Theory, Link Prediction, & Network Topologies
* **Core Algorithmic Architecture:** A dependency-free social network modeling system that visualizes network topologies through a real-time force-directed physics engine and generates contextual friend recommendations using a hybrid algorithmic approach.
* **The Hybrid Recommendation Formula:** Blends topological proximity and semantic interest overlapping to bypass the constraints of purely structural recommendation models:
  $$\text{Score}(u, c) = 0.6 \times \text{Jaccard}_{\text{Network}}(u, c) + 0.4 \times \text{Jaccard}_{\text{Interests}}(u, c)$$
  * *Network Topology Component (60%):* Evaluates structural proximity using Jaccard Similarity over mutual friend neighborhoods.
  * *Interest Similarity Component (40%):* Computes overlapping affinity across multiple color-coded interest vectors (e.g., Coding, Gaming, Music, Travel).
* **Data Structures & Algorithmic Paradigm:** Maps in-memory adjacency list representations via an optimized `HashMap<Integer, List<Edge>>` paired with metadata node tracking (`HashMap<Integer, UserNode>`). Discovers new connection candidates using Scalable Triadic Closure Expansion logic (friends-of-friends), maps shortest path routing via a Breadth-First Search (BFS) engine, and sorts final outputs through a high-efficiency Greedy Max-Heap Priority Queue to return the Top-$N$ recommendations in $O(K \log N)$ time.
